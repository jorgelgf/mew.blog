# BLOG - Meow.Blog
## Projeto de um blog com contos do universo felino 

## Tecnologias utilizadas:
ReactJS

<b>Para instalação e uso é necessário o NodeJS instalado no computador</b>

<p>Comandos para instalação e uso da aplicação:</p>

```
npm install;
npm start;
```
