# BLOG - Meow.Blog
## Project for a blog with tales from the feline universe

## Technologies used:
ReactJS

<b>For installation and use, NodeJS is required installed on the computer.</b>

<p>Commands for installing and using the application:</p>

```
npm install;
npm start;
```
